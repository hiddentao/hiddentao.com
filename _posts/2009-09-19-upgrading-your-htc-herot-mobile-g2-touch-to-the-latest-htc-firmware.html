I'd like to write about what I had to do to get my T-Mobile-branded (and sim-locked) phone upgraded to the latest HTC firmware. You see, HTC recently released an updated version of the firmware for the phone but [T-Mobile have been holding out](http://www.theregister.co.uk/2009/09/17/t_mobile_hero_update/) on releasing this update to their customers. You can't manually download the firmware from HTC and install it on your own since it checks the serial number of the phone and will only work if your phone is an unlocked, generic HTC Hero model. After some searching around the web I found a brilliant site - [http://android.modaco.com/](http://android.modaco.com/) - which provides a [customised ROM](http://android.modaco.com/content/htc-hero-hero-modaco-com/292018/08-09-2-2-the-modaco-custom-rom-is-here-2-versions-based-on-the-new-update/) (includes the latest firmware update as well as other desirable goodies) as well as [official HTC ROMs](http://android.modaco.com/content/htc-hero-hero-modaco-com/291942/03-09-hero-roms-radios-in-update-zip-format/). But to install these onto your phone you first of all need to *root* it, which essentially means giving yourself [superuser access](http://www.linuxselfhelp.com/linuxterms/index/root.html) to the phone. To do this I followed the instructions and video on this page: [http://theunlockr.com/2009/08/27/how-to-root-your-htc-hero-in-one-click/](http://theunlockr.com/2009/08/27/how-to-root-your-htc-hero-in-one-click/) {% endcapture %}{{ markdown | markdownify}} {% capture markdown %} What the previous step does is replace the flash memory updater on your phone with a custom version that ignores serial number restrictions. Once I'd performed this step I downloaded the custom ROM and followed the instructions on the following page: [http://theunlockr.com/2009/08/27/how-to-load-a-custom-rom-on-your-htc-hero/](http://theunlockr.com/2009/08/27/how-to-load-a-custom-rom-on-your-htc-hero/) {% endcapture %}{{ markdown | markdownify }} {% capture markdown %} By the way, don't expect any sympathy from T-Mobile if you damage your phone from doing the above. But I followed the steps as outlined and now I have had the latest firmware running on my phone for a few days with no hiccups so far. If you do run into the difficulties there is a very active forum on the ROM site where you'll be able to get help (although honestly, I don't think you'll have any problems). On the ROM site you'll also find updated firmware for the phone's radio transceiver. With that done now I just need to figure out what apps I'm going to make my phone :) {% endcapture %}{{ markdown | markdownify }}
